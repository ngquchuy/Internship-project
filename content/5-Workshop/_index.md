# Workshop: Building a Serverless Video-on-Demand Platform on AWS

## Overview

This workshop provides a step-by-step guide to implementing the **Serverless Video-on-Demand Platform on AWS**. The content follows the current project structure, beginning with an overview and prerequisite preparation, followed by backend development, implementation of the event-driven video processing pipeline, system testing, web application integration, project conclusion, and demonstration.

Throughout the workshop, each section describes the required configuration, implementation steps, AWS services involved, and the interactions between system components. The workshop is intended to provide a reproducible implementation process for building and evaluating the proposed Serverless Video-on-Demand architecture.
