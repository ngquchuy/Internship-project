This Worklog records my entire process of learning, hands-on practice, and project implementation during my internship at the company. The internship is clearly divided into two main phases:

1. **Foundational Learning Phase (Week 1 - Week 6):** Acquired core knowledge of cloud computing and basic infrastructure services of Amazon Web Services (AWS) through lecture series and hands-on labs from the "First Cloud Journey Bootcamp" playlist. This phase was conducted over 6 weeks, starting from **April 17, 2026** to **June 01, 2026**.
2. **Project Implementation Phase (Week 7 - Week 12):** In-depth research, architecture design, and building a practical application project on AWS to complete the graduation internship report.

Below is a summary and detailed links to the work accomplished each week:

**Week 1:** [AWS Fundamentals, AWS Console, and AWS CLI](1.1-Week1/) (04/17/2026 - 04/23/2026) - Got acquainted with the learning path, learned about Cloud Computing, AWS Global Infrastructure (Region, AZ, Edge Location), core service groups, and set up the AWS CLI environment.

**Week 2:** [IAM, Account Security, and Cost Management](1.2-Week2/) (04/24/2026 - 05/01/2026) - Deep dived into AWS IAM (Users, Groups, Roles, Policies), enabled MFA for account security, and configured cost management tools using AWS Budgets.

**Week 3:** [Networking with Amazon VPC](1.3-Week3/) (05/05/2026 - 05/11/2026) - Built an isolated virtual network with Amazon VPC, explored and configured Public/Private subnets, Route Tables, Internet Gateways, Security Groups, and Network ACLs.

**Week 4:** [Advanced Networking and CloudFormation](1.4-Week4/) (05/12/2026 - 05/18/2026) - Deployed NAT Gateway for one-way outbound connectivity in Private Subnets, explored VPC connection methods, and learned Infrastructure as Code (IaC) with AWS CloudFormation.

**Week 5:** [Compute, Storage, and Backup](1.5-Week5/) (05/19/2026 - 05/25/2026) - Practiced virtual server management with Amazon EC2 (AMIs, Instance Types, Key Pairs, Elastic IPs), used EBS & Amazon S3 storage services, and set up automated backup mechanisms with AWS Backup.

**Week 6:** [AWS Workshop, Architecture Design, and Summary](1.6-Week6/) (05/26/2026 - 06/01/2026) - Completed remaining labs on AWS Workshop, designed standardized architecture diagrams using Draw.io with the AWS Icon Set, summarized 6-week foundational knowledge, and finalized the project topic.

**Week 7:** [Project Topic Preparation and System Design](1.7-Week7/) (06/02/2026 - 06/08/2026) - Documentation research phase, defining business requirements, and sketching the overall system architecture diagram. Analyzed business requirements and selected a Serverless Architecture for the Video-on-Demand system.

**Week 8:** [System Architecture Design for Project](1.8-Week8/) (06/09/2026 - 06/15/2026) - Focused on research, detailed system architecture design, mapping data flows, and selecting optimal AWS services for the project. Researched the video processing pipeline: AWS Elemental MediaConvert.

**Week 9:** [Finalizing Architecture Diagrams for Project](1.9-Week9/) (06/16/2026 - 06/22/2026) - Continued reviewing and perfecting the system architecture design, clarifying security components, networking, and cost optimization before deployment. Developed backend logic, Lambda functions, and completed core business APIs.

**Week 10:** [Starting Project Deployment and Construction](1.10-Week10/) (06/23/2026 - 06/29/2026) - Built an automated video processing pipeline using S3, Lambda, and MediaConvert. Configured content storage and delivery through Amazon CloudFront and S3.

**Week 11:** [Developing and Integrating Project Services](1.11-Week11/) (06/30/2026 - 07/06/2026) - Programmed core application features and connected backend services with AWS Cloud services. Optimized the system security layer with AWS WAF and strict access controls. Tested operational capabilities of API Gateway and CloudFront.

**Week 12:** [Project Completion and Final Report](1.12-Week12/) (07/07/2026 - 07/13/2026) - Conducted comprehensive system testing, performance optimization, and completed the graduation internship report. Completed progress evaluation and feedback forms.
