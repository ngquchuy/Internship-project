### Week 6 Objectives:

- Learn fundamental database concepts on AWS.
- Grasp the roles of Amazon RDS, Amazon Aurora, Redshift, and ElastiCache.
- Practice deploying RDS database instances, connecting applications, and testing backup/restore operations.
- Learn and practice database migration from SQL Server/Oracle to Aurora MySQL.
- Learn about DynamoDB and serverless application design patterns with DynamoDB.
- Learn about data analytics services on AWS, such as Glue, Athena, Kinesis Data Analytics, and QuickSight.
- Practice building data pipelines encompassing ingestion, storage, cataloging, transformation, analysis, and visualization.
- Get familiar with CloudShell, SDK, Cloud9, and DataBrew in the data processing flow.
- Complete the main labs of Module 06 and Module 07.
- Know how to verify results and clean up resources after labs.
